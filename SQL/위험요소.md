### YMD 형식 나타내기
``` sql
select DATE_FORMAT(HIRE_YMD,'%Y-%m-%d') AS HIRE_YMD
from doctor
```

### NULL값을 특정 값으로 바꾸기
``` sql
SELECT PT_NAME, PT_NO, GEND_CD, AGE, IFNULL(TLNO,'NONE') AS TLNO
FROM PATIENT
```

### NULL값 찾기
``` sql
TLNO is not null
```

### 그룹 만들어서 계산
