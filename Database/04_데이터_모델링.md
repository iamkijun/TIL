## 데이터베이스 시스템

---

### 1. 데이터 모델링과 데이터 모델의 개념

데이터 모델링

- 현실세계에 존재하는 데이터를 컴퓨터 세계의 데이터베이스로 옮기는 변환과정

데이터 모델

- 데이터 구조, 연산, 제약조건으로 구성
- 현실 세계를 개념세계로 추상화했을 때 어떤 요소로 이루어져 있는지를 표현하는 개념적 구조



### 2. 개체-관계 모델

- 개체 (Entity)
  - 현실 세계에서 조직을 운영하는 데 꼭 필요한 사람이나 사물과 같이 구별되는 모든 것
  - 개체 타입
  - 개체 인스턴스
  

- 속성 (Attribue)
  - 개체가 가지고 있는 고유한 특성
  - 단일 값 속성
  - 다중 값 속성
  - 단순 속성
  - 복합 속성
  - 유도 속성
  - 널 속성
  - 키 속성

- 관계 (Relationship)
  - 개체와 개체가 맺고 있는 의미 있는 연관성
  - 1:1 관계
  - 1:N 관계
  - N:M 관계




### 3. 논리적 데이터 모델

논리적 데이터 모델

- 사용자가 생각하는 데이터베이스의 모습
- 관계 데이터 모델
- 계층 데이터 모델
- 네트워크 데이터 모델
